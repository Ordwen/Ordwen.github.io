Base de données


Objectif

Pour ce projet, il était demandé de concevoir une base de données pour un logiciel de gestion de stocks de vins.

Il fallait tout d'abord réaliser des maquettes du logiciel de gestion afin de déterminer les fonctionnalités importantes, puis créer un modèle conceptuel de données.
Ceci fait, il nous restait à évidemment créer la base de données, générer des scripts de remplissage, et lister l'ensemble des requêtes SQL nécessaires pour le logiciel.
